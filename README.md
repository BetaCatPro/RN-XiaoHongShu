## Environment

- Java 11.0.2
- Node 14.18.0
- React-Native 0.71.8 (低于 0.67 版本的 React Native 需要 JDK 1.8 版本)
- Android Studio : android-studio-2022.2.1.19-windows

## Usage

通过 Android Studio `sync` ./android 目录

#### 启动服务端

```shell
cd ./server

npm install

npm run dev
```

#### 启动客服端

在项目根目录下运行

```shell
npm install

npm run android
```
